# Tutorial: Addressing an envelope

A US envelope has a few parts.

## Recipient block

Write the recipient's address, for example:
4321 OAK ST
OAKTON MD 12345-6789

## Business mail

For a company, add the firm on its own line:
XYZ CORPORATION
500 Commerce Street, Columbus, OH 43215-0500.

## PO boxes

A PO box looks like PO BOX 482, Chicago, IL 60607-0482.

## Rural routes

Rural mail uses HC 72 BOX 293 A, Duluth, MN 55811-9702.

## Return address

Put your own address, e.g. 200 Sender Avenue, Madison, WI 53703-0200.
