# API Reference

## Authentication

Request an API key by emailing api-keys@developers.example.com. Keys are
issued within 24 hours.

## Rate limits

Need a higher limit? Write to limits+increase@developers.example.com with
your use case.

## Webhooks

Configure a callback URL. For debugging help, contact
webhooks@developers.example.com.
