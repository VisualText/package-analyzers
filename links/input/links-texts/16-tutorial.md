# Tutorial: Deploying a static site

We'll deploy to a few hosts.

## Option A: GitHub Pages

Push to a repo and enable Pages. Docs:
https://docs.github.com/en/pages/getting-started-with-github-pages.

## Option B: Netlify

Connect your repo at https://app.netlify.com/start. Build settings live at
https://docs.netlify.com/configure-builds/overview/.

## Option C: Cloudflare Pages

See https://developers.cloudflare.com/pages/.

## Verify

Check headers with https://securityheaders.com/?q=example.com and run
https://pagespeed.web.dev/?url=https://example.com.

Done! Share your URL, e.g. https://mysite.pages.dev/.
