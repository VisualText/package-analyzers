# Contributing

Thanks for your interest! Please read this first.

## Before you start

Read our [code of conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
and the [style guide](https://style.example.org/).

## Workflow

1. Fork from https://github.com/example/project.
2. Read the docs at https://docs.example.org/contributing.
3. Run CI locally; see https://ci.example.org/docs/local.

## Reporting bugs

File issues at https://github.com/example/project/issues/new?template=bug.md.
Security reports go to the form at https://example.org/security.

## Translations

Help translate at https://translate.example.org/projects/main/.
